OVSR

Welcome to GitHub Pages of the OVSR application.

This project is made by Dries Goossens @degoossez and Koen Daelman @KoenDaelman.

To download the repository: 
$ git clone https://github.com/degoossez/OVSR

Information

The application compares OpenCL and Renderscript on Android. The goal of the project is to compare speed and user friendliness. We have build-in some basic image filters and you are able to write your own OpenCL or Renderscript code in the text view. It's possible to build the code in the text view at run time. This way it's easy to test your own code. The written code and the output images are very easy to save with our application so the user can easily re-use his own code.

To help the users getting started, we also have a template you can use and it's also possible to read/edit the build-in image filters.
